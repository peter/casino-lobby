## Installation

```
yarn install
```

### Run Development Server

Make sure you have the Node API running first and then start the Vue app:

```
yarn serve
```

### Run Unit Tests

```
yarn test:unit
```

### Linting

```
yarn lint
```

## Styling and CSS
